Juju Exporter
=========
Juju exporter is a Prometheus exporter snap which collects machines' running status in all models under a Juju controller
[Juju Exporter](https://github.com/canonical/prometheus-juju-exporter)

Requirements
------------

snap
prometheus-client
confuse
asyncio
juju

Role Variables
--------------

| Name                  | Default                  | Description                                    |
| --------------------- | ------------------------ | ---------------------------------------------- |
| snap_channel          | "2.9/stable"             | Version to download                            |
