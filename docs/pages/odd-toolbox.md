## Carina Desktop

Your own remote Linux desktop! This option gives you a full graphical environment where you can run almost any installed Linux application.

The Carina Desktop unlocks access to powerful software that requires a graphical interface. Once your desktop session is running, you can launch these tools directly from a terminal window or the Applications menu.
