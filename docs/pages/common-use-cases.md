## Access to STAnford Research Repository (STARR)

The STAnford Research Repository, called [STARR](https://med.stanford.edu/starr-tools.html), enables access to clinical data for research purposes. Carina offers the compute environment compliant for high-risk data on which STARR data can be stored and analyzed.

STARR data is curated and managed by Research IT. As data custodians, the STARR team provides access to its datasets and tables, stored in a Google Cloud Project (GCP) compliant for high-risk data. Stanford researchers are able to apply for access to STARR and request a Carina environment to carry their analyses.

Specifically, researchers can query STARR datasets and tables, stored in RIT's cloud environment, and save their own data cuts in their own Carina projects (either on Carina On-Premise or Carina Cloud, currently supported on GCP).

## Access to Population Health Sciences data

The Center for Population Health Sciences provides access to population health datasets via their [PHS Data Portal](https://stanforddev.service-now.com/it_services%EF%B9%96id=sc_cat_item&sys_id=4d60831e1bacd010685d4377cc4bcb26.html). Access instructions can be found on [PHS' website](https://phsdocs.stanford.edu/computing-environment/quick-tour-phs-data-portal.html).

Clients can cut their data on the PHS Data Portal and export their data cuts to a Carina project.
