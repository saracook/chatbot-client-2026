Globus is a high-performance data-transfer platform. [Globus Documentation](https://globus.stanford.edu)

## Globus can be used to transfer NIH Controlled Access Data to and from Carina.

For more information, please see Stanford's [Globus Policies](https://globus.stanford.edu/policies.html).

## Globus cannot be used for High-Risk Data. 

This may change in the future; Stanford is working with Globus to find a solution for some High-Risk Data.
